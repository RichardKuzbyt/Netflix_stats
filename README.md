# Netflix_stats
Tableau dashboard containing various statistics about Netflix statistics. 
Fields include; 
  Total movies and TV shows by year; 
  Total movies and TV shows by country; 
  Top 10 genres; 
  Movies and TV shows distribution; 
  Ratings; 
  Descriptions; 
  Date added; 
  Duration; 
  Rating; 
  Release year; 
  Genre;
